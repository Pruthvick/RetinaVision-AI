# RetinaVision-AI
Multi-Label Retinal Disease Classification with Grad-CAM Explainability

## Features

- Multi-label retinal disease classification
- EfficientNetV2B0 deep learning model
- Grad-CAM explainability
- Interactive Gradio interface
- Cross-platform web application
- Accessible from desktop and Android/iOS mobile browsers

 Final project pipelines are in : efficientNetFinal.ipynb and gradioo.ipynb
